This is the public release repo for Linux-Nextmove. It is purely for release artifacts and contains no source code.

Note that the following artifacts are included with each release:
- nmt_install_x.x.x.tar.xz - for installing a system from scratch, typically used by Nextmove production
- nmt_update_x.x.x.tar.xz - for updating an existing system
- Source code (zip) - empty
- Source code (tar.gz) - empty

Most customers should use the update file and not the install file, unless otherwise specified.
